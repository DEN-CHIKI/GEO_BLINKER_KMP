# MapView (Компонент рендеринга карты)
**Функция:** Визуализация WebView/Native карты и маркеров устройств.
**Бизнес-логика:**
- Управление состоянием маркеров (активен/неактивен).
- Расчет границ (Bounds) для отображения всех устройств.
- Взаимодействие с JavaScript мостом (если используется WebView).
